# One Day Projects

This repository aims to have 31 projects or improvements of previous projects, completing 31 days or 1 month. It will have back-end but some can be made its front-end counterpart

## Rules

1. IA is allowed only to documentation
2. At least 1 pull request per day (no code holding)
3. Language and framework is free
4. Docker is needed for every project
5. 100% unit test coverage
